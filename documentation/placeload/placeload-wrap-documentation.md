### Wrapping

You can set multiple `<VPlaceload />` on a row. Simply wrap them in `<VPlaceloadWrap />`

<!--code-->

```vue
<template>
  <VPlaceloadWrap>
    <VPlaceload class="mx-2" />
    <VPlaceload class="mx-2" />
  </VPlaceloadWrap>
</template>
```

<!--/code-->
