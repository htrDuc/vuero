### Wrapping

You can set multiple `<VPlaceloadAvatar />` on a row.
Simply wrap them in `<VAvatarStack />`

<!--code-->

```vue
<template>
  <VAvatarStack>
    <VPlaceloadAvatar class="mx-1" />
    <VPlaceloadAvatar class="mx-1" />
    <VPlaceloadAvatar class="mx-1" />
  </VAvatarStack>
</template>
```
