### Line Icons

Vuero inputs are fully compatible with Line Icons.
Use the `icon` or `iconify` props on the `<VControl />`
component to show an icon.

<!--code-->

```vue
<template>
  <VField>
    <VControl icon="lnil lnil-briefcase">
      <input type="text" class="input" placeholder="Company" />
    </VControl>
  </VField>
</template>
```

<!--/code-->

<!--example-->

<VField>
  <VControl icon="lnil lnil-briefcase">
