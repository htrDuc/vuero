### Size

Control the size of `<VPlaceloadText />` component to match your interface.
You can set the `height` and the `width` properties according your needs

<!--code-->

```vue
<template>
  <VPlaceloadText :lines="5" width="75%" last-line-width="25%" />
</template>
```

<!--/code-->

<!--example-->

<VPlaceloadText :lines="5" width="75%" last-line-width="25%" />

<!--/example-->
